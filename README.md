# WooCommerce — Tegro Money Payment Gateway (first version)

A WordPress/WooCommerce payment gateway plugin for accepting payments via **Tegro Money**.

> ℹ️ This is an early version. For the current connector see **[woo-tegro-payment-gateway](https://github.com/UnitonApp/woo-tegro-payment-gateway)**.

## Install
Drop `wc-tegro.php` into your WooCommerce plugins, activate, and set your Shop ID + Secret Key. See `readme.txt`.
